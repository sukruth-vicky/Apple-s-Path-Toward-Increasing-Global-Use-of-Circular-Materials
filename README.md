Apple and the Global Path to Circular Materials
A Case Study Analysis Based on McKinsey & Company’s Work

by Sukruth M

Overview

This project presents my analytical breakdown of McKinsey & Company’s case study on Apple’s efforts to increase the global use of circular materials. 
The work focuses on strengthening supply chain transparency, supporting mining communities, and advancing sustainable resource management.
